# Portfolio Personal — Victor M. Cuevas

Portfolio personal desarrollado con Nuxt 3 y Tailwind CSS. Diseñado para mostrar proyectos, habilidades y experiencia de forma limpia y moderna.

## Tecnologías

- [Nuxt 3](https://nuxt.com/)
- [Tailwind CSS v3](https://tailwindcss.com/)
- TypeScript
- Iconsax
- Nuxt Google Fonts

## Instalación y uso local

### Requisitos previos

- [Node.js](https://nodejs.org/)
- [Git](https://git-scm.com/)
- [PNPM](https://pnpm.io/installation) (recomendado) o npm

### Clonar el repositorio

```bash
git clone https://github.com/VictorMCuevas/super-portfolio.git
cd super-portfolio
```

### Instalar dependencias

```bash
pnpm install
# o
npm install
```

### Ejecutar en desarrollo

```bash
pnpm dev
# o
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en el navegador.

### Build para producción

```bash
pnpm build
# o
npm run build
```
