Proyecto Memorión

¡Pon a prueba tu memoria!


El proyecto memorión consiste en la creación mediante HTML, CSS y JavaScript, de un juego de memoria buscando parejas de cartas. El juego consta de dos modos de juego:
  • Modo normal: Modo de juego en el que no se muestran las cartas, hay que buscar las parejas clicando en ellas. Una vez se encuentre la carta igual, las dos quedará volteadas. En este modo, se contabiliza el tiempo y el número de intentos.
  • Modo flash: Modo en el que se voltean todas las cartas al princio de la partida durante 5 segundos. Después hay que encontrar las parejas en el menor tiempo posible.

  • JavaScript
